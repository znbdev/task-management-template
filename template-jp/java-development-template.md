## 5. ⚠️ リスク、依存関係、バッファ (Risk & Buffer)

### 外部依存
*   **チーム依存**：
    *   フロントエンドからのAPI連携支援が必要
    *   DBA のデータベース設計案レビューが必要
    *   インフラチームのサーバーリソース/ドメイン設定が必要
*   **第三者依存**：
    *   XX サービスからの API ドキュメント提供待ち
    *   XX ミドルウェア（Redis/MQ/ES 等）の導入が必要

### 技術リスク
*   **性能リスク**：大量データクエリでレスポンスが遅くなる可能性
    *   *対策*：事前に SQL 最適化を実施、ページネーション/キャッシュ/非同期処理を検討
*   **互換性リスク**：新バージョンが旧機能に影響を与える可能性
    *   *対策*：十分な回帰テストを実施、後方互換APIを保持
*   **並行処理リスク**：高並行シーンでデータ不整合が発生する可能性
    *   *対策*：分散ロック/楽観的ロック/トランザクション分離を使用
*   **セキュリティリスク**：SQL インジェクション/XSS/権限超越アクセス
    *   *対策*：パラメータ検証、プリコンパイル SQL、権限インターセプター

### プロジェクトリスク
*   要件変更リスク：途中での要件調整による手戻り
    *   *対策*：事前に要件範囲を確認、変更はレビューフローを経由
*   進捗リスク：技術難点の克服時間が予想を超える
    *   *対策*：バッファ時間を確保、リスクを適時にエスカレーション

### バッファ机制
*   総見積もり工数に **15% - 20%** のリスク予備時間を追加することを推奨
*   現在のバッファ：**X.Xh**（総工数の X%）

## 💡 Java 開発ベストプラクティス

### レイヤーアーキテクチャ規範
*   **Controller レイヤー**：パラメータ受信、検証、レスポンス封装のみ担当、ビジネスロジックを含まない
*   **Service レイヤー**：コアビジネスロジック、トランザクション制御、Mapper と他の Service を呼び出す
*   **Mapper レイヤー**：データベース CRUD のみ担当、複雑な SQL は XML に記載
*   **クロスレイヤー呼び出し禁止**：Controller → Service → Mapper、スキップ不可

### 一般的な落とし穴と回避
*   ❌ **NullPointerException**：すべての外部入力は null チェック必須、Optional/Objects.requireNonNull を使用
*   ❌ **トランザクション失效**：@Transactional は public メソッドのみ有効、同一クラス内呼び出しはプロキシを経由しない
*   ❌ **ループクエリ**：バッチクエリでループ単回クエリを置換、IN 句または JOIN を使用
*   ❌ **大トランザクション**：トランザクション範囲は最小限に、トランザクション内でリモート API を呼び出さない
*   ❌ **ハードコーディング**：設定項目は application.yml に記載、定数は統一定義

### 性能最適化テクニック
*   ✅ **データベース**：適切なインデックス、SELECT * を避ける、ページネーションクエリ、読み書き分離
*   ✅ **キャッシュ**：ホットデータ Redis キャッシュ、キャッシュ穿透/雪崩/击穿に注意
*   ✅ **非同期**：非コアフローは非同期処理（@Async/メッセージキュー）
*   ✅ **コネクションプール**：HikariCP パラメータを適切設定（最大コネクション数/タイムアウト時間）

### デバッグテクニック
*   IDEA Debug モードでブレークポイントデバッグ
*   ログ確認時は ERROR/WARN レベルに注目
*   Arthas で本番環境の問題をオンライン診断（慎重に使用）
*   Postman/Apifox で API テスト

### 推奨ツールチェーン
*   **ビルドツール**：Maven / Gradle
*   **コード品質**：SonarQube / Checkstyle
*   **API ドキュメント**：Swagger / Knife4j / YApi
*   **テストフレームワーク**：JUnit 5 + Mockito
*   **ログフレームワーク**：SLF4J + Logback
*   **JSON 処理**：Jackson / Fastjson2
*   **ユーティリティライブラリ**：Lombok / Hutool / Apache Commons
