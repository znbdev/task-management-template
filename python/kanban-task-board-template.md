## 使用说明

1. **任务卡片**：每个卡片代表一个最小可交付的工作单元
2. **状态迁移**：🟩 → 🟨 → 🟦(可选) → ⬜ 或 ❌
3. **优先级定义**：
   - 🔴 P0-紧急：阻塞发版或影响核心流程
   - 🟠 P1-高：重要功能，本迭代必须完成
   - 🟡 P2-中：可有可无，有时间再做
   - 🔵 P3-低：优化项或远期规划
4. **WBS 跟踪**：每个卡片独立记录预估/实绩工时，便于偏差分析
5. **每日更新**：建议每天早晨更新看板状态，确保信息实时准确
